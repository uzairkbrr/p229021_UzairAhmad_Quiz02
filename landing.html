Quiz 02 - Web Development Assignment

Assignment ka Maqsad

Ek HTML form banana jo user ka naam, password aur email le kar database mein mehfooz kare. Password ki validation lazmi hai aur tamam comments Roman Urdu mein hone chahiyein.

🔒

Password ki Zarooraat

Password mein yeh sab hona lazmi hai:

  • Kam az kam 8 characters
  • Kam az kam 1 bara harf (A-Z)
  • Kam az kam 1 chhota harf (a-z)
  • Kam az kam 1 number (0-9)
  • Kam az kam 1 khaas nishan (!@#$%^&*)
💻

Technologies

Is project mein yeh technologies istemal hui hain:

  • HTML5 & CSS3 - Front-end design
  • JavaScript - Client-side validation
  • Node.js + Express - Server
  • SQLite - Database
🛡

Hifazati Khasusiyaat

Is project mein yeh hifazati iqdamaat hain:

  • Password PBKDF2 + SHA-512 se hash hota hai
  • SQL Injection se bachao (Prepared Statements)
  • Client aur Server dono par validation
  • Session-based authentication
